# كيفاش تشغل البروجي كامل ب Docker

1. فتح التيرمينال (cmd ولا PowerShell).
2. سير للملف ديال البروجي:
```powershell
cd C:\Users\ettao\Desktop\spacehub
```
3. شغل كلشي ب Docker:
```powershell
docker-compose up --build
```

- هادشي غادي يشغل ليك الباك، الفرونت، و الداتا بيس.
- من بعد، سير للبراوزر و دخل: [http://localhost](http://localhost)

إلى بغيتي توقف كلشي:
```powershell
docker-compose down
```

بالتوفيق! 🚀
